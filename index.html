Setup instructions
1

Create your Google Sheet

Open Google Sheets, create a new sheet called "Trip Log" with these column headers in row 1:

Date | Day | Start Time | User Name | Purpose | End Location | End Time | Remarks
2

Add the Apps Script

In Google Sheets: Extensions → Apps Script → paste this code:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName("Trip Log"); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.date, data.day, data.startTime, data.userName, data.purpose, data.endLocation, data.endTime, data.remarks ]); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); }
3

Deploy as Web App

In Apps Script: click Deploy → New deployment → Web App → set "Who has access" to Anyone → Deploy → copy the URL.

4

Paste the URL above

Go to the "Google Sheets connection" section above, paste the Web App URL, and tap Connect. Done!
